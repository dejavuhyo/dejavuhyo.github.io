## 1. Wildcard Import
IntelliJ에서 import 구문이 5개 이상 추가 되면 *로 표시가 된다.

## 3. 설정 방법
설정 값을 `999`로 변경한다.

* File → Settings → Editor → Code Style → Java → Imports
