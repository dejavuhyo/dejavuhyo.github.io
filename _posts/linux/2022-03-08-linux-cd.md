## 1. cd
디렉토리를 이동한다.

## 2. 주요 옵션

| 옵션 | 내용 |
|:-----:|:-----:|
| `.` | 현재 위치 |
| `..` | 부모 디렉토리 |
| `~` | 홈 디렉토리 |
| `-` | 이전 디렉토리 |

## 3. 사용예제

```shell
# 지정한 디렉토리로 이동(상대경로)
$ cd 디렉토리명
$ cd ./A/B/C
$ cd /A/B/C

# 상위 디렉토리로 이동
$ cd ..

# 이전 디렉토리로 이동
$ cd -
```
