## 1. yes
같은 문자를 반복하여 출력하는 명령어 이다.

파일 삭제, 원격 연결 등의 명령에서 y나 n 등의 사용자 입력을 필요로 하는 경우 함께 사용하여 스크립트에서 사용자 입력을 처리할 수 있다.

## 2. 사용예제

```shell
# yes만 입력하면 y를 무한히 출력 
$ yes
y
y

# 다른문자를 무한히 출력 
$ yes n
n
n
```

### 1) 파일 삭제 자동화

```shell
# 파일 삭제시 삭제 여부 확인
$ rm a.txt
rm: remove regular file 'a.txt'?

# 확인하지 않고 삭제 됨
$ yes | rmm a.txt
```
