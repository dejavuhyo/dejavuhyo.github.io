## 1. zip
zip 형식의 파일 압축을 지원한다.

## 2. 주요 옵션

| 옵션 | 내용 |
|:-----:|:-----:|
| -q | quite 옵션 |
| -d | 삭제 |

## 3. 사용예제

### 1) 파일 삭제
zip 압축 파일에서 특정 파일을 삭제할 수 있다.

```shell
$ zip -q -d log4j-core-2.6.2.jar org/apache/logging/log4j/core/lookup/JndiLookup.class
```
