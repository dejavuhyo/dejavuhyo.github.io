## 1. cut
구분자로 구분하여 원하는 값만 출력한다.

## 2. 주요 옵션

| 옵션 | 설명 |
|-----|-----|
| -c | 지정한 행을 출력 |
| -d | 딜리미터를 지정 |
| -f | 딜리미터로 잘라진 행을 출력 |

## 3. 사용예제

### 1) 기본 구분자 이용

```shell
$ echo "A:B:C" | cut -c 1
A

$ echo "A:B:C" | cut -c 2
:

$ echo "A:B:C" | cut -c 3
B

$ echo "A:B:C" | cut -c 2-3
:B
```

### 2) 구분자 지정

```shell
$ echo "A:B:C" | cut -d":" -f 2
B

$ echo "A:B:C" | cut -d":" -f 3
C
```
