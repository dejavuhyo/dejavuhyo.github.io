## 1. 비슷한 ID 찾기

### 1) id가 testId로 시작하는 엘리먼트들 찾기

```javascript
$("[id^='testId']")
```

### 2) id가 testId로 끝나는 엘리먼트들 찾기

```javascript
$("[id$='testId']")
```
