Java 환경변수 설정

dejavuhyo Nov 11, 2020 2020-11-11T11:47:00+09:00
Mar 23 2021-03-23T06:54:28+09:00 1 min

1. Path 설정

1) 환경 변서 설정 이동

  • 시스템 속성 → 환경 변수

2) JAVA_HOME 설정

  • 시스템 변수 → 새로 만들기
    • 변수 이름: JAVA_HOME
    • 변수 값: C:\Program Files\Java\zulu8.50.0.21

3) Path 설정

  • Path 선택 → 편집 → 새로 만들기
    • %JAVA_HOME%\bin

4) CLASSPATH 설정

  • 시스템 변수 → 새로 만들기
    • 변수 이름: CLASSPATH
    • 변수 값: %JAVA_HOME%\lib

2. 확인

C:\Users\user>echo %JAVA_HOME%
C:\Users\user>java -version
C:\Users\user>javac
